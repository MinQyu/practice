# React 코딩테스트 준비 과제

이 저장소는 React 코딩테스트를 준비하기 위한 실습 과제 모음입니다.

## 📋 과제 목록

### 문제 1: Todo 리스트 앱 (기본 CRUD)
**난이도**: ⭐⭐

**요구사항**:
- Todo 항목 추가 기능
- Todo 항목 삭제 기능
- Todo 항목 완료/미완료 토글 기능
- 완료된 Todo와 미완료 Todo 개수 표시
- 로컬 스토리지에 데이터 저장 (선택사항)

**구현 위치**: `src/problems/Problem1/TodoApp.tsx`

---

### 문제 2: 검색 및 필터링 기능
**난이도**: ⭐⭐⭐

**요구사항**:
- 사용자 목록을 검색할 수 있는 기능
- 이름, 이메일로 검색 가능
- 실시간 검색 (debounce 적용)
- 검색 결과 개수 표시
- 검색어가 없을 때 전체 목록 표시

**구현 위치**: `src/problems/Problem2/SearchApp.tsx`

**제공 데이터**: `src/problems/Problem2/mockData.ts`

---

### 문제 3: 폼 검증
**난이도**: ⭐⭐⭐

**요구사항**:
- 회원가입 폼 구현
- 필수 필드: 이름, 이메일, 비밀번호, 비밀번호 확인
- 실시간 유효성 검사
- 에러 메시지 표시
- 제출 버튼은 모든 필드가 유효할 때만 활성화
- 제출 시 콘솔에 폼 데이터 출력

**검증 규칙**:
- 이름: 2자 이상, 한글/영문만 허용
- 이메일: 유효한 이메일 형식
- 비밀번호: 8자 이상, 영문/숫자/특수문자 포함
- 비밀번호 확인: 비밀번호와 일치

**구현 위치**: `src/problems/Problem3/SignUpForm.tsx`

---

### 문제 4: API 호출 및 데이터 처리
**난이도**: ⭐⭐⭐⭐

**요구사항**:
- 외부 API를 호출하여 데이터를 가져오기
- 로딩 상태 표시
- 에러 처리
- 데이터를 카테고리별로 그룹화하여 표시
- 무한 스크롤 또는 페이지네이션 구현 (선택사항)

**API 엔드포인트**: 
- JSONPlaceholder API 사용: `https://jsonplaceholder.typicode.com/posts`
- 또는 제공된 mock API 함수 사용

**구현 위치**: `src/problems/Problem4/PostList.tsx`
